同时不受限于各大平台。

  1. 搭建环境准备（包括 node.js 和 git 环境，gitHub 账户的配置）

  2. 安装Hexo

一、搭建环境准备

  • Node.js 的安装和准备

  • Git 的安装和准备

  • gitHub 账户的配置

配置Node.js环境

下载Node.js安装文件：

Windows Installer 32-bit

Windows Installer 64-bit

保持默认设置即可，一路Next，安装很快就结束了。 然后我们检查一下是不是要求的组件都安装好了，同时按下 Win 和 R，打开运行窗口，输入：

node -v
显示v0.12.1

npm -v
显示2.5.1

如果正确，可以进行下一步了，如果不正确，则需要回头检查自己的安装过程。

配置Git环境

下载Git安装文件

Node.js 一样，大部分设置都只需要保持默认，Path选择第二项，一路Next。我们来检查一下Git是不是安装正确了，打开命令行，输入：

git –version
显示git version 1.9.5.msysgit.1

如果正确，可以进行下一步了，如果不正确，则需要回头检查自己的安装过程。
